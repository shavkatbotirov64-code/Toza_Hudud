# Smart Trash System - Admin Panel

Professional React-based admin panel for Smart Trash Management System.

## Features

- ✅ Modern React 18 with Hooks
- ✅ Real-time Dashboard with Statistics
- ✅ Interactive Map with Leaflet
- ✅ Bins Management
- ✅ Vehicles Tracking
- ✅ Routes Optimization
- ✅ Reports & Analytics
- ✅ Alerts System
- ✅ **Telegram Bot Integration** (NEW!)
- ✅ Settings & Theme Customization
- ✅ Responsive Design
- ✅ Professional UI/UX

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Telegram Bot Section

The new Telegram Bot section allows you to:
- View bot statistics (users, messages, commands)
- Monitor incoming messages from users
- Send broadcast messages
- View available bot commands
- Control bot status (active/inactive)
